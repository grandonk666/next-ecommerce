# Grandonk Merch Website

This is a ecommerce website build with NextJS and Strapi (headless CMS)

## Features

- Respomsive Layot
- Automated Payment

## Tech Stack

**Front-End:** React, NextJS, TailwindCSS

**Server:** Node, Strapi (headless CMS), Midtrans (Payment Gateway)
